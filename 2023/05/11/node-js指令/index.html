#node.js指令
##Node.js是一个基于Chrome JavaScript运行时建立的平台，用于开发服务端应用程序。在Node.js中，npm是Node.js的包管理器，可用于安装和管理依赖项。以下是有关Node.js的一些常用指令。
+node -v和npm -v：这些指令可用于验证Node.js和npm是否正确安装。如果成功安装，它们将显示其版本号。
+npm install ：这个指令用于在当前目录下安装一个包。可以将-g选项添加到指令中以进行全局安装。
+npm search ：这个指令用于搜索npm库中的包。
+npm install @：这个指令用于安装特定版本的包。
+npm list ：这个指令用于查看安装的包的版本信息。
+npm ls：这个指令用于查看当前目录下安装的所有包的版本信息。
+npm uninstall ：这个指令用于卸载一个包。
+npm init：这个指令用于创建一个package.json文件，该文件包含项目的配置信息和依赖项。
+npm install –save：这个指令用于在当前目录下安装一个包，并将该包添加到package.json文件的dependencies选项中。
+npm install –save-dev：这个指令用于在当前目录下安装一个开发阶段的工具包，并将该包添加到package.json文件的devDependencies选项中。

除了npm指令，Node.js还提供了一些命令行API，包括内置调试、多种执行脚本的方式和其他有用的运行时选项。要在终端中将Node.js的命令行API作为手册页查看，则运行man node。
